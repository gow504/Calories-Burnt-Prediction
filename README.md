# Calories-Burnt-Prediction
Machine Learning project that predicts calories burned during physical activity using user attributes and exercise-related data.

## Data preprocessing and exploratory data analysis (EDA)
Feature engineering and selection
Machine learning model training and evaluation
Performance comparison using regression metrics
Calorie burn prediction for new user inputs
Data visualization and insights generation

## Tech Stack:

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook

## Resume Project Description:

Developed a Machine Learning model to predict calories burned during physical activities using biometric and exercise-related features. Performed data preprocessing, exploratory data analysis, model training, and evaluation using Python and Scikit-learn to achieve accurate prediction results.
